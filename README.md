# openclaw-skill-phoenixd-wallet

OpenClaw skill for installing and operating ACINQ **phoenixd** (a Lightning wallet daemon) via **phoenix-cli**.

## Install / Use

Download the `.skill` from the latest release and import it into OpenClaw.

Releases: https://github.com/aenea251-cmyk/openclaw-skill-phoenixd-wallet/releases

## What’s inside

- `phoenixd-wallet/SKILL.md` — quickstart + common commands
- `phoenixd-wallet/scripts/download_phoenixd.sh` — helper to download and extract phoenixd binaries
- `phoenixd-wallet/references/systemd.md` — example systemd unit

## Notes

This repo is intentionally minimal: the canonical docs live in the skill’s `SKILL.md`.
